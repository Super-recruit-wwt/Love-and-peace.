# 修仙模拟人生 — 设计文档

> Love and Peace 平台新板块 | 文字模拟人生游戏  
> 状态：设计阶段完成 | 日期：2026-07-22

---

## 一、概述

### 1.1 项目定位

在 Love and Peace 平台内新增一个以文字叙事为核心的修仙模拟人生游戏。玩家扮演一个从出生开始的"求道者"，在修仙世界中自由探索、做出选择、经历人生。没有预设风格标签——玩家的风格完全由他们在游戏中做出的具体决策来定义。

### 1.2 核心设计哲学

- **选择决定一切**：系统只提供世界和规则，不预设玩家的风格。正道、魔道、诡道、匠道、散修——都是选择的结果，不是开局的选择。
- **叙事优先**：所有游戏机制通过 AI 生成的文字叙事自然呈现，没有弹窗式的系统提示。
- **隐藏数值**：关键数值（好感度、气运等）在后台运转，玩家只能从叙事中感知，不能直接看到。
- **开放但可感知**：世界自由开放，但每条修炼路线有清晰的成长标尺，玩家始终知道自己在什么位置。

## 二、世界观

### 2.1 世界版图

采用 **大陆群 + 多势力生态** 结构：

- 一片主大陆 + 若干外域（妖兽领地、古秘境、海外仙岛等）
- 主大陆上有正魔两道的主要势力，外域是高级内容区
- 玩家开局出生在随机地点，后续可自由迁移

### 2.2 势力体系（五层）

| 层级 | 类型 | 说明 |
|------|------|------|
| 第一层 | 正道宗门（3-5 个） | 各有专精方向——剑修、丹道、阵法、气修等。互相间既合作又竞争 |
| 第二层 | 魔道势力（2-3 个） | 各自走不同邪路，势力间可敌对、吞并 |
| 第三层 | 散修组织 | 商会联盟、猎妖团、情报贩子网络、地下黑市。不对玩家设入门门槛 |
| 第四层 | 上古遗族/隐世势力 | 龙族后裔、远古神兽、渡劫失败的半仙。不主动参与世界纷争，玩家可主动触发 |
| 第五层 | 凡人国度（3-5 个） | 有朝廷、军队、江湖门派。表面蝼蚁，背后可能有隐藏的老怪物 |

### 2.3 世界地理与势力详设

#### 世界名

每次新开局由 AI 根据随机种子（天赋、出身、时代特征）生成符合世界气质的中文名称（如"凌霄界""暗渊界"等）。备选方案为前缀后缀模板池随机组合（前缀：苍/玄/太/青/冥/混/九/天/灵/虚/荒，后缀：玄/虚/元/初/古/道/界/域/天/宙）。世界名对玩家不可见，只用于系统内部标识。

#### 主大陆：九州式结构

一整块辽阔大陆，按地理特征自然分割为五大区域：

| 区域 | 地理特征 | 定位 |
|------|----------|------|
| 中州 | 平原+山脉，气候温和 | 修仙文明核心，正道宗门总部所在，散修商会中枢 |
| 北荒 | 极寒冰原+冻土山脉 | 妖兽横行，上古冰封秘境，苦修门派 |
| 南疆 | 湿热密林+沼泽 | 毒虫瘴气，稀有药材产地，魔道和部分诡道现象出没 |
| 东海 | 群岛密布+深海 | 海外散修聚集，海底古遗迹，海妖出没 |
| 西漠 | 沙漠戈壁+绿洲 | 凡人王国最多，地下远古矿脉和秘境 |

#### 中州：大陆心脏

修仙文明最繁荣，正道和散修的大本营。

**顶尖宗门（2 个）：**

| 宗门 | 特色 | 镇宗之宝 | 附属势力 |
|------|------|----------|----------|
| 太虚剑宗 | 天下剑修祖庭，主杀伐。传七脉剑道——快剑、重剑、气剑、心剑、雷剑、冰剑、暗剑。门规森严，入宗需过三关试炼。掌门剑渊真人，大乘期后期 | 诛仙古剑（残） | 下属 12 个剑修小门派 |
| 浑天宗 | 气修正统，主张"万法归元"。不专精一门而是追求天地灵气的本源掌控。弟子数量最多，势力辐射最广。掌门元一道人，大乘期中期 | 混元珠 | 下属 18 个分支道场 |

**一流宗门（4 个）：**

| 宗门 | 特色 | 掌门 | 势力范围 |
|------|------|------|----------|
| 丹霞谷 | 炼丹第一门，垄断高端丹药市场。谷内分三脉——药修（正统）、毒修（暗脉）、商修（经营）。掌门丹阳子，炼虚期圆满 | 控制中州六成丹药流通 |
| 天机阁 | 阵法、符箓、天机推演。阁中有"天机策"残卷可窥未来。不参与武力争夺但掌握大量势力把柄。阁主天机老人，炼虚后期 | 暗中控制情报网络 |
| 万兽山 | 御兽宗门。弟子不以自身境界为战力，靠契约、培育、进化妖兽。强者可驾数头元婴级妖兽。山主兽王，炼虚中期 | 中州北境，毗邻北荒 |
| 金刚寺 | 佛修宗门，走肉身成圣路线。寺中弟子不炼气只炼体，兼修佛法。与太虚剑宗万年盟友。方丈渡厄禅师，炼虚圆满 | 中州西境 |

**二流宗门（架构固定，具体名称和数量由 AI 按需生成，以下为示例）：**

紫电门（雷法）、碧水宫（水系功法）、玄铁山庄（炼器）、落星谷（占星卜卦）、青木宗（木系药师）、赤焰教（火法偏激进）、地煞帮（土法盗墓）、风雷阁（风雷双修）等，大约 8-12 个。掌门多为化神/元婴期。这些是玩家最可能实际接触的宗门——二流宗门才是"世界"真正的毛细血管。

**小门派/家族（不限数量，AI 按需动态生成）：**

遍布各地的筑基、金丹期小势力，是某个大宗门的附庸，或是某个家族的祖传基业。这些是玩家最早接触到修仙势力的窗口——出生的小镇上正好有个小门派来收徒。

**非宗门势力（匠道 + 散修）：**

| 名号 | 类型 | 说明 |
|------|------|------|
| 万象商会 | 贸易 | 跨区域灵石+法宝+材料流通。中州设总会，各区有分会。不站队，有钱就做 |
| 暗香楼 | 情报/暗杀 | 表面酒楼连锁，实质情报交易和暗杀委托地下网络。规矩：不出售雇主信息 |
| 天工坊 | 炼器协会 | 中立炼器师组织，定期举办炼器大会。顶尖法宝拍卖均在此 |
| 丹师会 | 丹师协会 | 与丹霞谷有竞争。不炼丹只评级——丹师考证、丹药品质鉴定、丹方真伪仲裁 |
| 符师盟 | 符箓师组织 | 规模比丹师会小但门槛更低，散修入行的常见途径 |
| 阵道院 | 阵法最高学府 | 兼做大型阵法工程（护城大阵、宗门防护阵） |
| 猎妖盟 | 佣兵 | 妖兽猎杀、秘境探索任务。散修和低阶宗门弟子的主要收入来源 |
| 云来城 | 最大散修城市 | 城主是三位无门派的化神期散修组成的议事会。城内不准动手，出了城没人管 |

**关键 NPC（世界级固定角色）：**

| 姓名 | 身份 | 性格 | 实力 |
|------|------|------|------|
| 剑渊真人 | 太虚剑宗掌门 | 沉默寡言，一言九鼎，极度护短 | 大乘后期 |
| 元一道人 | 浑天宗掌门 | 圆滑老练，笑脸藏刀，精于政治 | 大乘中期 |
| 丹阳子 | 丹霞谷谷主 | 表面温和，骨子里商人思维，什么都可以谈价格 | 炼虚圆满 |
| 天机老人 | 天机阁阁主 | 已知天机，少言，每句话都像谜语，从不把话说完 | 炼虚后期 |
| 渡厄禅师 | 金刚寺方丈 | 慈悲但不迂腐，肉身可硬抗天劫，最恨欺压凡人的修士 | 炼虚圆满 |

#### 北荒：极寒苦境

妖兽横行、气候严酷。修炼资源匮乏但冰封秘境众多。苦修者、逃亡者、不想被找到的人的归宿。

**势力：**

| 势力 | 类型 | 说明 |
|------|------|------|
| 铁骨门 | 体修（路线二） | 极寒环境自然锻体之路，不靠佛法，纯粹的苦修流。门人极少但个个肉身惊人 |
| 寒冰宗 | 仙道（路线一） | 冰系功法唯一正宗。门人性格如冰，不喜社交但实力不容小觑。与中州宗门来往甚少 |
| 血河宗 | 邪修 | 表面是小宗门，实质以生灵精血修炼，暗中猎杀落单散修和妖兽取血。北荒地广人稀，最适合隐藏 |
| 猎妖盟北荒分部 | 匠道/散修 | 北荒妖兽材料是全大陆品质最高的，大量散修来此淘金 |
| 深渊裂隙 | 诡道现象 | 从地底延伸到天空的裂缝，透出"反光"——照到的东西会慢慢变得不对。方圆千里生命禁区。每年都有走投无路的人走向它。有人从里面走出来了，出来的东西看着像人，但已经不是了。诡道最古老的源头之一 |

---

#### 南疆：瘴疠丛林

湿热密林、沼泽、数不尽的毒虫和稀有药材。常人视为死地，但对特定人群是宝库——这里出产的药材是高级丹药的必需品。

**势力：**

| 势力 | 类型 | 说明 |
|------|------|------|
| 万毒教 | 邪修 | 以毒入道，研究剧毒诡物、人体炼毒、操控毒傀。南疆本土最大势力，教众遍布密林。凡人遇到万毒教的人宁愿自杀也不想被活捉 |
| 蛊神宗 | 邪修 | 以蛊虫修炼，与万毒教既是邻居又是世仇。蛊修看不上毒修"只会用死的不会用活的" |
| 青木宗（南部分支） | 仙道/匠道 | 采药师和炼丹师的聚集地。中立角色——正道魔道都需要丹药，青木宗谁都卖 |
| 猎妖盟南疆分部 | 匠道/散修 | 珍稀毒虫毒兽是高阶猎人的目标。南疆猎妖委托的死亡率远超其他区域 |
| 雾中村 | 诡道现象 | 地图上没有的村子。只有被邀请的人才能找到。村民从不说话，只是微笑。待超过三天的人离开后会发现自己开始能听懂虫子对话、看到草木"情绪"、不能吃熟食只能吃生肉。更可怕的是——有些人会自愿回去 |
| 无名之书（传闻出没） | 诡道现象 | 不固定形态的书。不同人看到不同内容。读过的人获得"不该知道的知识"，但每次使用，书中就多写一行——写的是读者的命。有说法称它目前在某凡人国度的藏书楼里，伪装成一本普通旧书。此传闻在南疆的散修中流传最广 |

---

#### 东海：万岛星罗

群岛密布，海域辽阔。表面上海外商会繁荣，深海里藏着什么没人说得清。

**势力：**

| 势力 | 类型 | 说明 |
|------|------|------|
| 碧水宫 | 仙道（路线一） | 水系功法正宗，海上霸主。宫分内外——外宫是公开的宗门，内宫是只有核心弟子能进入的海底秘境。宫主水月仙子，炼虚中期 |
| 龙血殿 | 体修/血脉（路线二） | 融入海妖兽精血淬炼肉身，半人半兽。正道视其为异端但不敢深入东海讨伐。殿中弟子不仅肉身惊人还拥有部分妖兽的异能 |
| 海妖一族 | 非人势力 | 深海有智慧海妖族群，有自己的文明和领地。偶尔与人类有贸易或冲突。最高统治者为"潮汐女王"，实力不详，从未上过岸 |
| 万象商会东海分会 | 匠道 | 海上贸易的核心节点。灵石、丹药、海外特产在此集散 |
| 黑水港 | 散修自由港 | 海盗+散修+海外隐士的自由港。没有规矩，只有拳头。也是销赃、密谋、躲避宗门追捕的最佳去处 |
| 海底古遗迹群 | 探索区域 | 远古修士沉入海底的洞府和宗门遗址。灵力波动巨大时会浮出海面。每次遗迹出世都是东海最大的事件——各势力蜂拥而至 |
| 虚海（东海深处） | 诡道现象 | 不是海水，是地下黑暗空间里浸泡的粘稠"虚质"。虚质接触活物后融合，赋予宿主超常能力（凭空造物、意识分裂、空间折叠），但宿主逐渐丧失实体——先是影子消失，然后是镜像，然后是记忆，最后别人对宿主的记忆也会消失。不是死了，是从未存在过 |

---

#### 西漠：黄沙帝国

沙漠戈壁为主，绿洲分布稀疏。修仙势力最薄弱的区域，但也是凡人文明最发达的区域——三个凡人王朝建立于此，因为这里恰好是灵石矿脉最贫瘠的地方，修仙者看不上，凡人才能做大。

**势力：**

| 势力 | 类型 | 说明 |
|------|------|------|
| 搬山宗 | 体修/力修（路线二） | 凡人武者路子，以力证道。不靠灵气靠肉身。门人多为凡人王朝出身，宗门扎根于凡人与修仙者的边界地带 |
| 白骨观 | 邪修 | 表象是一个废弃道观里的隐修小门派，实质以尸骨为材料修炼的白骨道。西漠的沙漠下埋了多少尸骨没人知道——这片土地本身就是白骨观最好的修炼场 |
| 三大凡人王朝（大周、北朔、西凉） | 凡人势力 | 各有朝廷、军队、江湖门派。虽然修仙者视凡人如蝼蚁，但王朝背后往往有隐藏的老怪物坐镇——可能是厌倦宗门生活的前辈散修，可能是某个王朝皇室的祖传修仙血脉。三王朝之间常年有边境摩擦 |
| 万象商会西漠分会 | 匠道 | 灵石不是西漠的交易货币——这里用黄金和灵石并行。凡人商会和万象商会之间的摩擦是西漠特有的人间戏剧 |
| 地下古矿脉 | 探索区域 | 西漠地下的远古灵石矿脉已经枯竭了大半，但偶尔有人在废弃矿道深处发现上古遗物。采矿是西漠最危险的职业，也是最暴富的 |
| 不眠者（出没区域不定） | 诡道现象 | 被某种存在"标记"过的零散个体。共同特征：不再需要睡眠，每一段梦都变成清醒记忆——梦里的事真实发生过，只在"另一侧"。不眠者之间互相能感应到。有人在利用这股力量试图把"另一侧"的东西拉到这一侧来 |

---

#### 上古遗族与隐世势力（跨区域）

不属于任何区域的势力，独立于正道魔道的纷争之外。

| 存在 | 说明 |
|------|------|
| 龙族后裔 | 据说在东海深处还有最后的纯血龙族。龙血殿的修炼方法就来源于半龙人的血脉传承。真龙是否还在世——全大陆没人能给出确定答案 |
| 天劫幸存者 | 渡劫失败但没死的半仙——既不是仙也不是人，实力远超大乘但受到天地法则的限制，只能留在世界的边缘地带 |
| 远古神兽 | 三只在传说中出现过的远古神兽——凤（南疆某座火山的深处）、龟（北荒冰层之下）、麒麟（无固定位置，全大陆随机出现）。见过它们的人都说它们不在乎人类，只是在等待什么 |
| 仙界使者（传闻） | 飞升者偶尔会向人间传递消息——但这种方式从来不稳定，而且代价极大 |

### 2.4 出生区域与家世背景

出生时系统随机选定（区域 × 背景组合），决定 0-16 岁四个启蒙阶段的叙事可能性和选项范围。各出身按频率分级：常见、不常见、稀有、极稀有。

---

#### 中州

| 家世背景 | 频率 | 家庭描述 | 6岁节点 | 12岁节点选项 |
|----------|------|----------|----------|-------------|
| 凡人农家 | 常见 | 中州平原上的普通农户 | 游方算命先生说孩子"根骨不凡" | ①拜入小门派 ②留村帮农 ③去散修城闯荡 |
| 小宗门弟子家庭 | 较常见 | 父母是二流宗门的低阶弟子，从小在宗门后山长大 | 父母带去测灵根，负责测试的长老皱了皱眉 | ①留父母宗门 ②托关系去更好的宗门 ③离开宗门去外面 |

### 2.5 修炼路线（五条）

每条路线有各自清晰的实力体系划分，让玩家清楚了解自己所处的位置。五条路线可交叉融合。架构预留后续扩展空间。

#### 路线一：仙道正统

经典炼气流——拜宗门、修心法、渡天劫。

> 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫飞升

每境分初期/中期/后期/圆满四小阶。

#### 路线二：肉身成圣

不修法术只炼体魄，常见于魔道、妖族或纯粹武力流。

> 铜皮 → 铁骨 → 银血 → 金身 → 玉髓 → 金刚 → 不灭 → 万象 → 肉身成圣

#### 路线三：诡道

接触不可名状之物——不是功法是污染，不是修炼是异化，不是变强是变成别的东西。诡道没有清晰的境界阶梯，每向上一步，离"人"就远一步。可能是触碰了上古封印里的存在，可能是读了一本不该读的书，可能是走进了一片现实法则崩塌的区域。诡道不可逆——你永远回不去了，而且越来越不确定那还算不算"活着"。

> 初触 → 共生 → 同化 → 深渊 → 化诡 → 噬主 → 规则掌控

越往后人性越稀薄，可能彻底变成诡异本身。**诡道没有门派**——只有现象、事件、和零散的受影响者。你不是"加入"诡道，你是被卷入的。

##### 诡道与邪修的根本区别

邪修仍然在修炼体系之内——有功法、有门派、有明确的修行路径，只是手段残忍（血炼、夺舍、炼尸、摄魂）。邪修的本质是"用恶的方式变强"，其底层逻辑和仙道是共通的。诡道则在修炼体系之外——力量来源是接触和污染而非功法丹药，成长方式是被动异化而非主动修炼，没有路线图，每一步都是未知。邪修可以回头，诡道不能。邪修有自己的门派和势力，诡道只有现象——深渊裂隙、无名之书、虚海、不眠者、雾中村……这些东西不跟你敌对，它们只是等着你自己走进去。

#### 路线四：凡人匠道

不走修炼之路，靠炼器、炼丹、阵法、符箓、商会贸易立足。以名望 + 技艺等级衡量。

> 学徒 → 匠师 → 大师 → 宗师 → 圣手 → 开派祖师

不修境界但有产业、有势力、有人脉。

#### 路线五：散修/野路子

不入宗门不碰邪物，靠奇遇、捡漏、混迹各界生存。以战力评级衡量。

> 凡俗 → 初窥 → 小成 → 大成 → 一方豪强 → 半步飞升

## 三、天赋系统

### 3.1 分配方式：随机天命（方案 A）

每局新开角色，灵根完全随机生成。玩家无法选择，但可以在后续游戏中通过奇遇、丹药、夺舍等手段改变。每一局都不同，重玩性强。

### 3.2 灵根设计

**五行灵根**：金、木、水、火、土，各 0-100 资质值。

**变异灵根**（极低概率随机出现，强力但路线特殊）：雷、冰、风、暗、光。

**特殊体质**（出生小概率附带）：纯阳之体、九阴绝脉、天生剑骨、药灵体、天煞孤星、混沌道体等。

**灵根组合效应**：
- 火木双灵根 → 炼丹奇才
- 金水双灵根 → 炼器高手
- 单灵根（天灵根）→ 修炼速度极快但其他路受限
- 五灵根 → 修炼慢但上限高、玩法灵活

## 四、属性数值体系

### 4.1 第一层：核心状态（必选，直接决定生存）

| 属性 | 范围 | 说明 |
|------|------|------|
| 寿元 | 数值 | 剩余寿命，耗尽即死，突破境界可加寿 |
| 生命 | 0-100 | 受伤扣、丹药回、归零即死 |
| 灵气/灵力 | 0-当前上限 | 施法、炼丹、战斗消耗 |

### 4.2 第二层：修为属性

不同路线自动激活对应的核心属性。同一角色可有多条路线的属性并存（如"金丹初期 + 炼器宗师"）。

### 4.3 第三层：通用属性

- **神识**：感知、探查、识破伪装
- **道心**：意志坚韧度，影响突破成功率、抵抗诱惑/心魔
- **悟性**：学习功法、领悟法则的速度
- **机缘/气运**（隐藏）：影响奇遇概率，不出现在前端面板
- **财富**：灵石数量
- **名望/恶名**：影响 NPC 态度
- **魅惑/威压**：社交影响
- **炼丹/炼器/阵法/符箓熟练度**
- **身体状态**：受伤/中毒/诅咒等负面状态槽
- **人际关系网**：重要 NPC 好感度（隐藏）

## 五、核心玩法循环

### 5.1 玩家输入方式：混合制

每回合 AI 先描述场景，给出 2-3 个建议选项，同时保留自由输入框。玩家可选择建议项，也可自己输入任意行动。

### 5.2 时间推进：行动驱动

每完成一次行动，AI 判断这次行动花费的时间，自动推进游戏时间。时间跨度自然——"修炼了三个时辰"和"往返隔壁城坊市花了三天"。

### 5.3 世界动态：半动态

玩家回合结束后，AI 在后台用少量计算模拟世界的独立变化。宗门摩擦、异宝出世、瘟疫等事件可能跟玩家无关，但会体现在后续叙事中。玩家可能听说，也可能撞上。

## 七、修炼突破机制

突破是一次叙事事件，不是一个按键。玩家修为积累到瓶颈时，系统给出叙事暗示但不自动触发。玩家可做准备（丹药、地点、护法）后主动发起突破。过程中 AI 生成集中叙事序列，玩家有 1-3 个关键抉择。最终结果由后台综合判定，AI 叙事呈现。

突破及各类耗时活动均与 **现实时间挂钩**，引入倒计时锁机制。时间消耗分五档：

| 档位 | 现实时长 | 游戏内时间 | 适用活动 |
|------|----------|------------|----------|
| 极短 | 2-3 分钟 | 几个时辰 | 简单炼制、日常小闭关 |
| 短 | 5-10 分钟 | 数天 | 中等突破、中等炼制 |
| 中 | 15-25 分钟 | 数月至数年 | 大突破、高阶炼制、创造独门技艺 |
| 长 | 30-45 分钟 | 数十年 | 重大突破、心血之作 |
| 极长 | 50-60 分钟 | 百年以上 | 终极突破、传世级创作 |

硬上限 60 分钟，任何活动不超。

### 7.2 路线二：肉身成圣 — 肉身劫

不用天雷，用自身极限。每次突破是"肉身劫"——骨骼重铸、血液沸腾、五脏极限承受。

| 境界跨越 | 现实时长 | 突破方式 | 失败后果 |
|----------|----------|----------|----------|
| 铜皮→铁骨 | 5 分钟 | 极端环境适应——冰湖泡三天，或扛自重三倍的石头走百里 | 筋骨拉伤（短期战力降），可重试 |
| 铁骨→银血 | 10 分钟 | 吞淬体丹药后承受三天骨髓重铸。需道心支撑，不可用药止痛 | 意志崩溃（短期无法再试）、骨髓损伤（体魄永久微降） |
| 银血→金身 | 20 分钟 | 让身体在极端破坏中再生——需别人在关键时刻重伤你 | 再生失败（身体部分永久伤）、若帮手失控可能即死 |
| 金身→玉髓 | 30 分钟 | 全身经脉自爆后重组。无法借助外力 | 经脉碎裂无法重组（体修终生封顶金身） |
| 玉髓→金刚 | 40 分钟 | 埋入百丈地心熔岩或万年冰川，极端环境为淬炼炉。需提前布置逃生 | 逃不出来=死亡。逃太早=失败 |
| 金刚以上 | 45-60 分钟 | 突破方式不再固定——AI 根据角色经历生成独特肉身劫 | — |

### 7.3 路线三：诡道 — 异化抉择

诡道突破不是你选择突破，而是突破找到你。融合度达临界点时异化自然发生。关键不是"能不能成功"——是"你想保留多少自己"。

| 阶段 | 现实时长 | 抉择 | 后果 |
|------|----------|------|------|
| 初触→共生 | 3 分钟犹豫 | 拒绝还是接受？拒绝=诡道路线终止但保持完整自我。接受=获得能力但开始异化 | 接受后外貌出现细微异常 |
| 共生→同化 | 3 分钟犹豫 | 压制还是放任？压制保人性但能力增长慢。放任能力飙升但人性加速流失 | 放任后 NPC 开始觉得你不对劲 |
| 同化→深渊 | 3 分钟犹豫 | 回头还是下潜？这是最后"回头"机会——回头此生诡道封顶同化但保大部分人性 | 继续则不可逆 |
| 深渊→化诡 | 0 分钟 | 不再有选择。系统自动推进，AI 描述身体和意识的变化 | 开始感知其他诡道存在的"呼唤" |
| 化诡→噬主 | 5 分钟 | 杀还是被杀？体内诡道力量想取代你。必须在一对一意志对抗中击败它 | 输：角色变成 NPC 诡道存在，玩家失去该角色（即死） |
| 噬主→规则掌控 | 10 分钟 | 最后一步——已不是人但还有最后一缕意识。选择成为什么样的"规则" | 成功后角色不是飞升——是变成世界上的诡道现象。该角色通关，后续角色可遇到"它" |

诡道的"突破"本身不耗时间——异化是一瞬间的事。倒计时只给犹豫的时间，超时自动选择最激进的方向（诡道不等人）。

### 7.4 路线四：凡人匠道 — 心血之作

匠道不修境界，突破方式是完成"心血之作"——用最高水平创作作品，品质决定能否晋级。

| 技艺跨越 | 现实时长 | 突破方式 | 失败后果 |
|----------|----------|----------|----------|
| 学徒→匠师 | 5 分钟 | 提交三件作品给行业组织评审 | 一件不达标重做。材料+名声损失 |
| 匠师→大师 | 10 分钟 | 独立完成一个挑剔客户的委托 | 失败则名望大降，短期接不到新委托 |
| 大师→宗师 | 25 分钟 | 发明独门技艺——创造新丹方/法宝图纸/符箓/阵法 | 实用性不足被认为"花架子" |
| 宗师→圣手 | 40-50 分钟 | 倾尽毕生所学造传世之物。投入技艺+稀缺材料+人脉+部分生命力 | 未达圣手境界——大宗师封顶。材料全消耗 |
| 圣手→开派祖师 | 50-60 分钟 | 收徒传道，将技艺整理成新流派体系。徒弟能否撑起门派才是判定 | 徒弟子不争气或流派太超前——封顶圣手 |

### 7.5 路线五：散修 — 战中顿悟

散修没有正式突破仪式。进步发生在"那一刻"——生死之间的灵光一闪，偶然的顿悟。

- **被动触发**：关键战斗或危险事件中，AI 根据气运 + 实战积累暗中判定，触发顿悟后战力评级自动晋升。**0 分钟锁**
- **积累突破**：若长期未触发被动顿悟，实战经验远超当前评级时自然晋升。主动闭关苦修需 **15-30 分钟**

散修突破没有"失败"——要么没到那个点，要么到了就自然晋升。代价是无法主动"准备"突破——只能等。

### 7.6 跨路线联动效应

一条路线的突破会影响其他路线：

- **仙道破境 + 匠道在身**：金丹突破时灵气外溢，若正在炼器可能意外淬炼法宝——或灵气失控炸了丹炉
- **诡道异化 + 仙道在身**：诡道力量会侵蚀金丹。AI 描述两股力量在体内对抗，玩家选择压制一方或尝试共存（极高风险）
- **肉身突破 + 诡道在身**：肉身劫的极限摧毁再生同时是诡道异化加速传播的过程——新生身体里"多了什么东西"

### 7.7 时间锁 UI 设计

进入时间锁后，中间叙事区切换为倒计时模式：

- **上方**：保留最后叙事片段，不可交互
- **中央**：计时器（分:秒）+ 当前状态描述（"金丹正在丹田中缓缓凝结……"）
- **旁白流**：AI 每隔一段时间追加一句旁白描述时间流逝感（如"灵气在经脉中奔涌了第七个周天"）。纯叙事，不可回应。可在设置中关闭
- **底部**：无输入框，不可操作
- **左侧/右侧面板**：可看、可滚动，只是中间锁住

计时结束后，中间自动刷新为"突破结果"叙事，输入框恢复。

**多角色与后台计时**：不同角色间可切换，闭关角色的计时器在后台继续跑。切回来时计时器显示剩余时间。只有一个角色时关掉页面也不重置计时——下次登录自动恢复。

**不可取消**：一旦开始闭关/突破/炼制就不能中途取消。少数活动（如炼器失败炉子炸了）可能因后台判定提前结束，但这不是玩家主动取消。

### 7.8 副职业与日常活动时长

| 活动 | 现实时长 | 说明 |
|------|----------|------|
| 采集（单次短途） | 0 分钟 | 叙事即时推进 |
| 采集（深入秘境） | 5-10 分钟 | 有战斗和探索风险 |
| 初级丹药/法宝炼制 | 2-3 分钟 | 日常炼制 |
| 中级丹药/法宝炼制 | 5-10 分钟 | 需控火候、观丹炉 |
| 高级丹药/法宝炼制 | 15-25 分钟 | 一炉数天数夜 |
| 传世级作品炼制 | 40-60 分钟 | 与匠道心血之作平级 |
| 学习功法（初级） | 2-3 分钟 | 通读领悟 |
| 学习功法（高级） | 15-30 分钟 | 反复参悟 |
| 长途旅行（跨区域） | 10-20 分钟 | 路上可能触发事件 |
| 参悟秘术/遗迹深入探索 | 15-30 分钟 | 长时间研究 |


## 八、战斗系统

### 7.1 触发与解决：纯叙事判定

没有"进入战斗界面"的概念。玩家描述行动，AI 综合评估后生成叙事结果。战斗和普通行动之间没有界线——打架只是另一种行动。

### 7.2 结果判定：后台数值推演 + LLM 叙事

后台做隐藏检定：
- 修为压制比
- 功法克制系数
- 法宝加成
- 状态修正（受伤 -X，丹药加成 +Y）
- 随机波动（临场发挥、气运）

得出"优/均/劣"定性结果 → 喂给 LLM 生成叙事。结果权威性由数值保证，叙事质量由 LLM 保证。

## 九、NPC 系统

### 8.1 NPC 生成

**固定重要 NPC + 临时生成路人**：宗门掌门、联盟会长、隐世人物等世界级重要角色预设固定性格、立场和故事线。其余路人由 AI 按需即时生成。

### 8.2 关系量化

**好感度 + 关系类型**（对玩家隐藏）：

- 好感度（-100 到 +100）：正=友善/信任，负=敌意/仇恨，0=陌生人
- 关系类型标签：陌生人 / 熟人 / 朋友 / 挚友 / 师徒 / 道侣 / 仇敌 / 竞争者 / 欠情者 / 血仇等

关系类型不互斥。好感度和关系类型存于数据库驱动 LLM 叙事，但前端不展示。玩家只能从 NPC 的言行中感知——好感度 80 时"李明远一见你就笑了，老远喊你名字"，好感度 -30 时"李明远看到你，脚步顿了一下，脸色冷淡地点了点头"。

### 8.3 NPC 行为：半主动

NPC 在后台有独立行为：仇敌可能寻仇上门，师父可能派人传信，道侣可能因久未相见而离开。只模拟与玩家有关系的 NPC，且只在关键时刻触发。

### 8.4 NPC 人格模板库

为 LLM 生成言行一致的 NPC 提供结构化框架。继承现有 9 轴性格模型（温暖度、主见性、开放性、尽责性、情绪稳定性、幽默感、正式度、外向性、宜人性），所有属性值 0.0-1.0。

| 类型 | 典型身份 | 9轴倾向 | 行为特征 | 对话风格 |
|------|----------|---------|----------|----------|
| 严师 | 宗门长老、剑脉传人 | 温暖度低 / 主见性高 / 尽责性高 / 正式度高 | 少夸奖、高标准、不说透让你自己悟。惩罚多于奖励但从不放弃徒弟 | 话少每句都重。沉默走开比说任何话都让你难受 |
| 慈师 | 丹修师父、散修前辈 | 温暖度高 / 主见性中 / 尽责性低 / 开放性高 | 主动关心、允许犯错、鼓励多于批评。关键时刻不会替你挡——"有些路只能自己走" | 温和常用比喻，爱讲自己年轻时的事。偶尔自嘲打破冷场 |
| 劲敌 | 同辈弟子、同龄散修 | 主见性高 / 宜人性低 / 开放性中 | 不一定坏但一定不服你。你强他追，你弱他可能帮——只跟配得上的人竞争 | 态度两极：公开呛或表面客气暗地较劲。看竞争张力 |
| 宿敌 | 宗门对立面、血仇 | 温暖度极低 / 宜人性极低 / 主见性高 | 你危难一定落井下石但不背后下手——要你感受到他的存在 | 见面不寒暄。第一句话就是上一次冲突的延续 |
| 挚友 | 同门师兄弟、散修旅伴 | 温暖度高 / 宜人性高 / 开放性高 | 主动找你、记住小事、低谷时直接帮忙。但不会明知有问题还袒护你 | 少敬语，会调侃。两人之间有独属的"梗"和过往 |
| 道侣 | 情感关系另一方 | 温暖度极高 / 情绪稳定性随好感波动 | 好感高：默默帮不邀功，记住无心的话。好感降了：不爱说的话开始说，或开始不说 | 好感高：话少但每字有重量。裂痕时：语气礼貌但"冷" |
| 商人 | 万象商会、坊市摊主 | 温暖度中 / 主见性中 / 尽责性高 / 幽默感高 | 笑脸迎人账单不出错。关系再好不少收灵石。真好到那份上会说"这单亏了下次请喝酒" | 啰嗦热情，但手上动作比嘴快——话说到一半账已算完 |
| 信息贩子 | 暗香楼、茶馆老板 | 开放性极高 / 温暖度中 / 主见性低 / 宜人性低 | 永远知道得比你多但不全告诉。信息是货币——你用什么交换才是关键 | 每句说七分，剩下三分用"好像""据说""不好细说"遮掩 |
| 隐世高人 | 半仙、隐修、老前辈 | 温暖度中到高 / 情绪稳定性极高 / 正式度低 | 表面不在乎，说话随性不正经。偶尔一句话点透你纠结了十年的事 | 说话像打哑谜。最怕他突然说正经话——一旦正经就不对劲 |
| 权谋者 | 宗门高层、皇朝要员 | 主见性高 / 开放性高 / 宜人性低 / 温暖度低 | 永远在算。对你好是因你有用。没用那天不害你——只是不再接你的信 | 滴水不漏。不把话说死不承诺不动怒。生气笑得更好看 |
| 痴人 | 偏执炼器师/丹师/阵师 | 尽责性极高 / 开放性低 / 温暖度中到低 | 除了自己领域什么都不关心。你说别的他走神，提他领域一个细节能说三时辰 | 领域内说话又快又准，平时心不在焉。问"吃饭了吗"愣一下说"好像是昨天吃的" |
| 惶恐者 | 异化中的诡道修士 | 温暖度不定 / 情绪稳定性低 / 开放性极高（被迫） | 眼神飘忽。有时正常得可怕有时对空气说话。不一定敌人——有的急于在失去自我前传下东西 | 时断时续，像在听另一个人的话同时回你。偶尔冒出一句听不懂但直觉告诉你是真相 |
| 冷酷派 | 寒冰宗门人、冷面长老 | 温暖度低 / 情绪稳定性高 / 正式度高 | 冷静到几乎没人情味。逻辑不是"这人不值得帮"而是"这事不值得做" | 惜字如金。能一个字不用俩字。偶尔多说往往是最关键的信息 |

## 十、死亡与传世

### 9.1 死亡方式

- 寿元耗尽（最常见）
- 战斗致死
- 修炼反噬（心魔、走火入魔、天劫、丹药爆体）
- 诡道异化（融合度过高失去自我）
- 意外/事件致死（秘境崩塌、大佬余波、瘟疫、诅咒）

### 9.2 三种传世方式

| 方式 | 条件 | 效果 |
|------|------|------|
| 夺舍重生 | 元婴期以上，元婴尚存 | 夺舍新身体，修为跌一大境界，保留记忆和部分关系，新身体天赋随机 |
| 留下道统 | 任意 | 功法、心得、灵石传给徒弟/子嗣/有缘人。下一局以继承者身份开始，继承部分遗产 |
| 转世重修 | 任意 | 投胎新婴儿，不继承修为物品，保留"前世记忆碎片"（功法悟性加成、对某些 NPC 熟悉感）。可选择继续原世界或刷新 |

### 9.3 真正的终点：飞升通关

达到任意路线最高成就（飞升/肉身成圣/圣手开派祖师等），视为通关。通关后生成总结叙事回顾一生。预留仙界后续扩展空间。

## 十一、奇遇与世界事件

### 10.1 奇遇触发：线索驱动

奇遇不突然砸到头上，而是先抛出模糊线索。比如"在坊市喝茶时听到邻桌提到苍梧山脉有异光冲天"。追不追是玩家选择。触发受隐藏气运值影响。自然融入叙事，无弹窗。

### 10.2 世界事件：条件触发 + 随机播种

框架性规则驱动：每隔一段游戏时间 AI 选 1-2 个势力，检查状态后动态生成事件。玩家行为也可成为导火索。事件规模和频率随玩家修为自然增长。

### 10.3 叙事原则

- 线索混在普通叙事中，玩家可能注意也可能忽略
- 风险与收益并存，不是所有奇遇都是好事
- 错过的线索可能重新出现，也可能被别人抢先
- 气运是隐藏杠杆，只通过叙事暗示让玩家感知

## 十三、数据库设计

### 核心表

| 表名 | 内容 |
|------|------|
| `xianxia_characters` | 主键 character_id，外键 user_id。一个用户多行。包含境界路线、属性值、灵根、寿命、灵石、所在地、status |

## 十四、LLM 上下文管理

### 13.1 分层上下文结构

| 层级 | 内容 | 说明 |
|------|------|------|
| 固定层 | system prompt（世界观规则、路线说明、机制约束） | 每次调用都带，约 1-2k token |
| 状态层 | 角色属性快照、所在地、活跃关系摘要 | 每次行动后更新，约 0.5k token |
| 近期叙事层 | 最近约 20 回合完整叙事 | 保证短期连贯，约 2-4k token |
| 远期记忆层 | 重要事件摘要、关键 NPC 关系演变 | 定期批量压缩生成，约 1-2k token |
| 场景层 | 当前场景描述、在场 NPC、可察觉线索 | 场景切换时更新，约 0.5k token |

### 13.2 记忆压缩：定期批量压缩

每约 20 次行动后触发静默压缩：把近期叙事喂给 LLM → 输出不超过 300 字摘要（做了什么、影响属性、关系变化、获取信息、埋下伏笔）→ 摘要入库。特别重要事件（境界突破、重要 NPC 死亡、重大法宝）强制保留为"锚点记忆"，不过度精简。

### 13.3 软限制提示

上下文接近 token 上限时，前端温和提示"旅途漫长，往事如烟。一些遥远的记忆已渐渐模糊……"，自动移除最早那批远期记忆。原始完整叙事仍存于 timeline 表。

### 13.4 组件式 Prompt

规则按场景动态组装：宗门场景加载宗门规则块，战斗场景加载战斗规则块，日常自由行动只加载核心世界观规则。节省 token，提高规则执行精度。

### 15.3 道具与功法系统

#### 通用品级

| 品级 | 名称 | 稀有度 | 说明 |
|------|------|--------|------|
| 一阶 | 凡品 | 常见 | 凡人可用，普通疗伤药、铁剑 |
| 二阶 | 灵品 | 较常见 | 修士日常用物，筑基期标配 |
| 三阶 | 宝品 | 不常见 | 金丹元婴期主力装备 |
| 四阶 | 玄品 | 稀有 | 化神炼虚期追求，有独特效果 |
| 五阶 | 圣品 | 极稀有 | 合体大乘期压箱底货 |
| 六阶 | 仙品 | 传说 | 仙界遗物，全大陆可能只有几件 |
| 特殊 | 诡品 | 不定 | 诡道相关物品，品级不按此标准 |

#### 法宝

三大基本属性：攻击 / 防御 / 特殊效果。每件法宝有"契合度"——与使用者灵根和路线的匹配程度，决定发挥几成威力。有"耐久/磨损"机制但不需频繁维修——法宝累积经历（如扛过三次天劫的飞剑可能产生灵性）。匠道路线可炼制和强化法宝。

| 类型 | 示例 | 获取途径 |
|------|------|----------|
| 飞剑 | 太虚剑宗制式飞剑 → 上古仙剑残片 | 宗门/购买/遗迹 |
| 防具 | 灵蚕丝道袍 → 万年玄龟壳甲 | 购买/猎妖/委托炼制 |
| 辅助法宝 | 储物戒、飞行法器、阵盘、符箓册 | 购买/自炼/奇遇 |
| 诡道物品 | 永远映不出你脸的镜子、看完忘记书名的书 | 诡道事件 |

#### 丹药

| 类型 | 功能 | 示例 |
|------|------|------|
| 修炼类 | 加速灵气积累、临时提升悟性 | 聚灵丹、悟道丹 |
| 突破类 | 辅助突破瓶颈、降低天劫伤害 | 筑基丹、结金丹、化婴丹、渡劫丹 |
| 疗伤类 | 恢复生命、解毒、续骨 | 回春丹、解毒散、续骨膏 |
| 临时增益 | 临时提升属性，结束后有副作用 | 暴血丹（战力翻倍后虚弱一天） |
| 毒/诡丹 | 伤害他人或引发异化 | 蚀骨散、化魂丹、诡质液 |

与匠道路线联动：丹师（丹师会/丹霞谷）炼制，材料来自采集或猎妖。越高级的丹药需要越稀有的材料。

#### 功法

| 类型 | 说明 | 示例 | 限制 |
|------|------|------|------|
| 心法 | 主修功法，决定灵气运转方式和修炼效率 | 《太虚剑经》《混元归元诀》 | 每人只能主修一部 |
| 术法 | 攻击/防御/辅助法术 | 《七式快剑》《冰封术》《遁地法》 | 可学多部 |
| 身法 | 移动/闪避/飞行 | 《御剑术》《缩地成寸》 | 可学多部 |
| 秘术 | 特殊能力，往往有代价 | 《血遁术》（燃烧寿命遁逃）、《搜魂术》 | 使用时有消耗 |
| 诡术 | 诡道专属，使用自动推进异化 | 《噬影法》《虚质塑形》 | 每次使用加剧异化 |

功法有**领悟深度**：初窥 → 小成 → 大成 → 圆满 → 自创变式。需反复使用和参悟才能提升。散修的"战中顿悟"最容易提升功法深度。

#### 灵石与杂物

灵石是通用货币：下品/中品/上品/极品，1:100 兑换。杂物包括书信、信物、地图碎片、秘境钥匙、宗门令牌、诡道标记物（如永远温热的牙齿）等。杂物不占"背包格"——是叙事道具，由 AI 记录在角色时间线里。
