# 凡人修仙传 - 第十一卷「真仙降世」事件提取

## 事件一（第2171-2187章）

**起因**：韩立闭关冲击大乘瓶颈

**行动**：
- 先修炼敖啸老祖所传神念秘术至大成，压制银月体内忘情诀反噬
- 消耗九成异魔金汲取晶珠能量，三年内法力达到合体后期大圆满
- 以极品血牙米修炼百脉炼宝诀，四十年苦修至大成
- 身躯各部分化为法宝级存在，举手投足具撼动天地之力
- 炼制第三座极山（元合五极山之一）
- 为度大乘天劫做最后准备

**地点**：荒野山脉洞府

**结果**：
- 百脉炼宝诀大成
- 第三座极山炼成
- 合体后期大圆满

**后续影响**：准备冲击大乘

## 事件二（第2178-2187章）

**起因**：五行灵潮降临，韩立引发大乘天劫

**行动**：
- 以元合五极山和青竹蜂云剑阵对抗五行之劫
- 渡真雷劫——以雷纹凝练之术化解天雷
- 黑白雷球劫——噬金虫群起吞噬天雷
- 魔劫——辟邪神雷化为金色电网退魔
- 心魔之劫——以炼神术和大衍诀斩杀心魔
- 进阶大乘成功，天地异象降世

**地点**：荒野山脉

**结果**：
- 韩立渡过天劫，进阶大乘期
- 六翼霜蚣在雷劫中进化
- 噬金虫群在雷劫中蜕变

**后续影响**：韩立成为灵界大乘至尊

## 事件三（第2188-2197章）

**起因**：韩立进阶大乘后首次出行，寻找六翼霜蚣踪迹

**行动**：
- 神游万里寻踪
- 遇见樊咆子，受邀参加道果大会
- 返回人族，圣岛使者前来朝拜
- 韩立翻脸惩戒嚣张的圣岛使者杜宇
- 一击而溃——大乘之威震慑宵小

**地点**：人族 → 圣岛

**结果**：
- 威名传遍人妖两族
- 圣岛低头

**后续影响**：韩立确立大乘威势

## 事件四（第2198-2204章）

**起因**：韩立举办大乘庆典，万修来朝

- 庆典开始，各族使者齐贺

- 黑枭王（敌对大乘）来犯挑衅

- 斩杀大乘——韩立以雷霆手段斩杀黑枭王
- 威震诸族——无人敢再挑战

**地点**：韩立洞府 → 庆典会场

**结果**：

## 事件九（第2245-2305章）

- 旧敌出现——魔族大军卷土重来
- 秘藏现——傀儡大军遗迹

**地点**：魔界 → 广源斋 → 傀儡遗迹

**结果**：
- 噬金虫王诞生（灵界最强灵虫之一）

**后续影响**：韩立力量不断增长，准备对抗真仙

## 事件十五（第2445-2446章）

**起因**：韩立修炼圆满，引动飞升天劫

**行动**：
- 炼神术第三层大成
- 五藏锻元功大成

- 熔岩之雨——以银色巨鸟（冰凤变身）的奇寒之力冻结

- 仙界之门——乳白色裂缝打开

**结果**：
- 渡过飞升之劫
- 被仙界接引之光带入仙界
- 醒来已身在北寒仙域飞仙台
- 接引人高升告知：欢迎来到北寒仙域

**后续影响**：韩立正式飞升仙界，全书完结
